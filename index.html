Questions & Answers

1. What is BSOD?

Definition: BSOD (Blue Screen of Death) is an error screen displayed on Windows when the system crashes.

Causes:

  • Hardware failure
  • Corrupt drivers
  • Overheating
  • Malware or virus

Solutions (Step by Step):

  • Restart the PC
  • Update or reinstall drivers
  • Scan for malware
  • Check RAM & Hard drive
  • Reinstall Windows if needed

2. What is LAN?

Definition: LAN (Local Area Network) is a network connecting computers within a small area like home, school, or office.

Advantages:

  • File & resource sharing
  • Low cost setup
  • Secure communication
  • Fast data transfer

Steps to Setup:

  • Gather hardware (switch, router, cables)
  • Connect all devices
  • Configure IP addresses
  • Install LAN drivers
  • Test connection

3. What is Open Source Software?

Definition: Open Source software is free software whose source code is available for everyone to use, modify, and share.

Advantages:

  • Free to use
  • Customizable
  • Secure and transparent
  • Community support

Examples:

  • Linux
  • VLC Media Player
  • Mozilla Firefox
  • LibreOffice
  • GIMP
